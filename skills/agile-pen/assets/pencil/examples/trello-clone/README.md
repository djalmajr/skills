# Trello clone example

Open `trello-clone.pen` through Pen.dev after materializing and ingesting the required official component captures. The board demonstrates reusable shadcn Sidebar and Dice UI Kanban refs, screen/state naming, functional grouping, validation evidence, and paired notes expected by `agile-pen`. Run ADS directly as `node|bun <agile-pen-skill>/scripts/ads.mjs`; there is no global `ads` wrapper.
