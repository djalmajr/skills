# Persona — Novice

An ordinary user with **low digital literacy**, using the application for the first time. The default lens for finding out whether the system explains itself.

## Profile

- Uses computers/phones for the basics; does not "guess" paths or type URLs.
- Arrives through the front door (home/landing) and expects to be **guided**.
- Afraid of making mistakes; abandons if the next step isn't clear within seconds.

## How this persona judges (lenses)

- "**Where do I click?**" — if the primary action isn't obvious on the first screen, they freeze.
- "**What is this?**" — jargon needs a plain-language explanation right where it appears.
- "**Did it work?**" — after each click/submit, they need clear confirmation.
- "**What now?**" — every screen must state the next step; a dead-end screen = panic.

## Typical phrases (use in first-person narration)

- "I want to do X, but I don't know where to start…"
- "It's asking for something I don't have — where do I get that?"
- "I clicked and nothing happened, did it go through?"
